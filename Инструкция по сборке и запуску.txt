1) Проект может быть собран в Visual Studio 2012 и всех последующих, с применением платформы .Net Framework 4.6.1
2) Для реализации базы данных необходим Ms Sql Server 2017. Необходимо запустить скрипт CreateAll.sql, либо скрипты по отдельности, расположенные в корневом каталоге в последовательности:
	а) 1)Create db.sql
	б) 2)Create tables
	в) 3)DocSumByPositionSumTrigger.sql и 4)Create Login - DbUser - Grant-Deny.sql в любом порядке.
3) Запустить проект для сборки и запуска.


В ПО реализована возможность заносить, хранить, редактировать и удалять информацию о документах и позициях этих документов. Весь необходимый фунционал представлен на форме. Также есть возможность добавлять и удалять элементы таблиц, и открывать детализацию документов, по щелчку правой кнопки мыши и выбору в выпадающем меню соответствующего пункта, удалять при помощи клавиши [Delete] на клавиатуре. 

В приложении реализован паттерн MVP.
Решение состоит из четырёх проектов.
1) AppModel - слой данных(Model)
2) AppPresenter - слой логики(Presenter)
3) DocInfApp - слой отображения(View)
4) SharedLib - библиотека, разделяемая между всеми слоями и имеющая классы объектов Document и Position, а также интерфейсы
 IPresenter, IView

В проекте внедрена слабая связь между DocInfApp и AppPresenter с помощью Managed Extensibility Framework.
